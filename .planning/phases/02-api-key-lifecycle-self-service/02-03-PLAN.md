## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-02-08 | Tampering (CSRF) | POST self-mint/rotate/revoke via session cookie | high | mitigate | Double-submit CSRF token (crypto/rand cookie + echoed header, constant-time compare) on top of SameSite=Lax + JSON preflight (D-09) |
| T-02-09 | Elevation (latency) | Stale privilege on long-lived key | medium | mitigate | Mandatory expiry with tight 180d max cap (D-08) enforced from this config in the handler |
| T-02-10 | Information Disclosure | CSRF token timing side-channel | low | accept | Constant-time compare used; token is not itself a secret credential |
</threat_model>
