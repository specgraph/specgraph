# Naming Alternatives: SpecGraph Rename

> **Status**: Under consideration
> **Date**: 2026-04-09
> **Reason**: Trademark concern requires renaming. New name must not be mystical. "Spec" and "graph" are allowed but not required.

## TL;DR

After evaluating 25+ candidates across domains (.io/.dev/.com), npm, crates.io,
Go module paths, GitHub orgs, and USPTO TESS, four finalists remain:

| | **plexograph** | **nascgraph** | **draftwright** | **chartergraph** |
|---|---|---|---|---|
| **Root** | Latin *plexus* (braided, woven) | Irish *nasc* (link, bond) | English *draft* + *wright* (maker) | English *charter* (foundational doc) + *graph* |

| **Story** | "Specs woven into braided structure" | "The graph that binds design together" | "The craftsman of design drafts" | "A graph of foundational charters" |

| **Short form** | `plexo` or `pxg` | `nasc` or `ng` | `dw` | `cg` or `charter` |
| **Domains** | All 3 free | All 3 free | .io/.dev free, .com expired (may drop) | .io/.dev free, .com parked ($500-$5k est.) |
| **Registries** | All clear | All clear | All clear | All clear |

---

## Evaluation Criteria

Each candidate was checked against:

- **Domain registration**: .io, .dev, .com — free vs. parked (with asking price) vs. active business
- **npm**: Package name collision
- **crates.io**: Crate name collision
- **Go module path**: `github.com/<name>/<name>` availability
- **GitHub org/user**: Namespace availability
- **USPTO TESS**: Live trademarks in software/IT classes 009 and 042
- **General web presence**: Existing businesses, products, or projects using the name

**Pass criteria**: Clean GitHub org, at least one of .io/.dev free or parked under $500, no npm/crates collision, no live software trademark.
