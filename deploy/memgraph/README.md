# Memgraph Deployment Profiles

Memgraph is an **in-memory graph database** — the entire dataset must fit in RAM.
These profiles provide pre-sized Docker Compose configurations for different scales.

## Profiles

| Profile | Engineers | Applications | Specs | Memory Limit | Docker Memory |
|---------|-----------|-------------|-------|-------------|---------------|
| **local-dev** | 1-5 | 1-3 | < 500 | 128 MB | 192 MB |
| **small-org** | 10-50 | 5-20 | 500-5K | 512 MB | 640 MB |
| **medium-org** | 200-500 | 50-200 | 5K-50K | 2 GB | 2.5 GB |
| **large-org** | 1K-4K+ | 500-2K+ | 100K-500K | 16 GB | 20 GB |

## Usage

```bash
# Local development (default — also generated by `specgraph init`)
docker compose -f deploy/memgraph/local-dev/docker-compose.yaml up -d

# Small organization
docker compose -f deploy/memgraph/small-org/docker-compose.yaml up -d

# Tear down and remove data
docker compose -f deploy/memgraph/<profile>/docker-compose.yaml down -v
```

## Persistence

All profiles enable WAL (write-ahead log) and periodic snapshots:

- **WAL** records every transaction for crash recovery
- **Snapshots** are periodic full-state dumps (interval varies by profile)
- **Snapshot-on-exit** ensures clean state on graceful shutdown
- Data lives in the `specgraph-data` Docker volume

To start fresh, add `-v` when tearing down to remove the volume.

## Large Organization Guidance

At 1000+ engineers / 500+ applications, a single Memgraph instance will work
but benefits from operational care:

1. **Shard by domain** — run separate instances per business domain or portfolio
2. **Archive aggressively** — export and purge superseded/done specs older than N months
3. **Add indexes** — `CREATE INDEX ON :Spec(slug); CREATE INDEX ON :Decision(slug);`
4. **Monitor** — Memgraph Lab on port 7444 (exposed in medium/large profiles)
5. **Consider Enterprise** — Memgraph Enterprise adds replication and HA

## Sizing Formula

Per-node memory (approximate):

| Entity | Base Size | With Authoring Data |
|--------|-----------|-------------------|
| Spec | ~1 KB | 5-20 KB |
| Decision | ~1.5 KB | ~1.5 KB |
| Constitution | 2-10 KB | 2-10 KB |
| Edge | ~80 bytes | ~80 bytes |

Total estimate: `(specs * 10KB + decisions * 1.5KB + edges * 80B) * 2` (2x for Memgraph overhead).
